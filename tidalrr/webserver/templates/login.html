2. Add a Tidal URL

Copy a Tidal URL.
It can be an artist's page, an album, a track or a playlist.
All content imported this way will be queued to download automaticaly and be monitored for future new content.

3. Scan the monitored items

Manually start the scan of monitored content (artists, albums, playlists), to add new (not downloaded) content to the download queue.
it will start automaticaly between 23:00 and 3:00 AM (configurable in the settings).

4. Download the queued files

Manually start the download of the queued content (tracks, album covers).
it will start automaticaly between 3:00 and 11:00 AM (configurable in the settings).
